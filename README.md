# FOOD-Pos-SYSTEM

FoodPanda POS is a lightweight web-based system designed for restaurant partners to interact with an online food delivery platform. The system provides a promotional landing page, feature highlights, pricing tiers, a contact form with validation, dark/light mode support, and a thank-you confirmation page.
This project demonstrates Agile workflow, JIRA-based epics & stories, Git branching, team collaboration, and deployment.

# Features Included

## Frontend UI Components

Hero section with promotional messaging

Feature section outlining system capabilities

Pricing plans with three service tiers

Contact form with input validation

Footer with social links

Responsive layout & modern styling

Dark Mode Toggle

Pop-up Signup Modal

## Backend Component

contact.php form submission handler

Error handling & user feedback

Logging simulated ticket messages

## Deployment & Integration

Git repository with feature branches

JIRA tracking of epics, stories & tasks

Web server deployment link

## Project Setup Instructions
****Prerequisites

Make sure  following things should be installed:

✔ PHP (any version above 7.x)
✔ A local server (XAMPP, WAMP, MAMP or built-in PHP server) ; recommeded XAMPP
✔ Git installed
✔ Web browser

## How to Run the Project Locally
* Step 1: Clone the Repository
git clone https://github.com/https://github.com/ahmadijaz02/Food-POS-System.git.git

* Step 2: Move into the Project Directory
cd FoodPanda

* Step 3: Start Local PHP Server
php -S localhost:8000

* Step 4: Open in Browser
http://localhost:8000

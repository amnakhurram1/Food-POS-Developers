# FoodPOS Landing Page – Responsive Restaurant Partner Website

A modern, responsive FoodPOS landing page inspired by the FoodPanda Partner Hub. This project was developed using PHP, HTML5, CSS3, and JavaScript to demonstrate responsive web design, frontend UI/UX principles, client-side interactivity, and basic backend form processing.

The application provides a professional landing page for restaurant partners with sections for features, pricing, contact, partner registration, dark mode support, and responsive layouts while implementing server-side form validation using PHP.

---

## Project Overview

This project simulates a commercial restaurant partnership landing page where restaurants can learn about platform services, explore pricing plans, register as partners, and submit inquiries through a validated contact form.

The project emphasizes clean UI design, responsive layouts, user experience, accessibility, and lightweight backend functionality.

# Form Validation

The contact form performs:

- Empty field validation
- Email format validation
- Session error handling
- Secure input sanitization
- Success redirection

---

# Backend Workflow

1. User submits contact form
2. PHP validates inputs
3. Invalid data redirects back with errors
4. Valid submissions are logged
5. User is redirected to Thank You page
